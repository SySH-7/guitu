# 归途 v0.2.1

这是 `0.2.1` 的正式签名修订构建，Android `versionCode` 为 `5`。

## 本次修复

- 书影页面默认年份与快捷年份改为读取设备系统时间，不再固定为 2026。
- 导入数据中即使包含未来年份，也不会改变默认年度趋势范围。
- 新增记录时禁止选择未来日期。
- 取消数据导出后不再错误提示导出成功。
- 删除单条记录前增加二次确认。
- 缺少地图数据时显示明确的“地图数据未配置”状态。
- 增加自动化测试、GitHub Actions 和项目截图。
- Android 发布包改用正式私有密钥签名。

## 地图数据说明

公开仓库和本 Release 中的公开 APK 不包含第三方 GeoJSON，也不包含私有地图数据。除地图展示外，阅读、影视、记录、统计、主题切换和 JSON 数据管理功能可正常使用。

个人完整版仅供作者本地保存和测试，不应上传到 GitHub、网盘或其他公开渠道。

## 旧版升级注意

早期上传的 `0.2.1` APK 使用过调试签名，不能直接覆盖安装本次正式签名版本。已有数据的用户应先在旧版中导出 JSON，确认文件保存成功后卸载旧版，再安装本次 APK 并导入 JSON。

请妥善保存本次使用的 keystore。后续所有正式版本必须继续使用同一密钥签名，Android 才允许覆盖升级。
